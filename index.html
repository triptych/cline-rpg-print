Printable RPG Game Creator

Game Parameters

Dungeon Map

Game Pieces

Room Cards

Game Rules

Character Stats

  • Health Points (HP): 6
  • Attack Bonus: +1
  • Defense: 7

Basic Mechanics

  1. Move one room per turn
  2. Combat: Roll 2d6 + Attack Bonus vs Monster Defense
  3. Trap Check: Roll 2d6 when entering a new room
    • 2-6: Trap triggers
    • 7-12: Safe passage

Victory Conditions

  • Collect required number of treasures
  • Find and reach the exit
  • Survive!
